会员简介

1、为什么江湖要推出会员制？
    作为一个江湖网站，租用一台稳定高速的服务器来存放日夜不停止运作的江湖，其年租价都在1000元以上，加上站长长期的上网费用和软件开发的时间精力和经费都是非常之高的，所以希望大家都来支持本站的发展，从中也可以得到更高的娱乐性节省泡点时间，希望大家能给予支持和理解！

2、会员制对站方和会员双方的得益？
    对于站方来说，收取的会费提供了租用更好、速度更快的服务器和开发更具有娱乐性的程序提供了必要的经费，有利于江湖的长期发展，对于会员来说比普通玩家拥有更多更有娱乐性的、更多的功能和减少泡点的时间，最简单一个事例：在广州地区网吧的上网费是5元/小时，而江湖会费一个月10元，对于在网吧上网的玩家来说这10元只是多上了两个小时的网，而加入会员后泡点却是普通玩家的1.5倍，实际上是省了泡点的时间节约了上网费用，所以说推出会员制双方都有得益！
