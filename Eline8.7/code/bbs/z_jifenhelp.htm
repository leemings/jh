积分帮助

============= 功能简介 =============

1. 根据用户在线时间计算财产值、经验
值、魅力值的增加值，加到相应属性项中
2. 提供后台管理功能，管理方便      

============= 积分规则 =============

在线 T 分钟以上才可以保存积分
保存积分的时间间隔为 T 分钟
系统不提供自动保存积分功能
积分保存后在线时间重零开始计算

============= 计算方法 =============

算法一: 一般算法

财产值 = 在线时间 / W
魅力值 = 在线时间 / C
经验值 = 在线时间 / E

算法二: 四舍五入算法

财产值 = (在线时间+5) / W
魅力值 = (在线时间+5) / C
经验值 = (在线时间+5) / E

算法三: 四舍五入、最小值不小于1算法

财产值 = MAX（(在线时间+5) / W ，1）
魅力值 = MAX（(在线时间+5) / C ，1）
经验值 = MAX（(在线时间+5) / E ，1）

其中，T,W,C,E 由管理员在后台设定
